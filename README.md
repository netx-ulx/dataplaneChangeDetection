## Getting Started

We used python 3.8.5 to run this application. Other versions of python should run this solution but have not been tested.

### Prerequisites

The following libraries must be installed to run this program.
* scapy
```sh
pip3 install scapy
```
* mmh3
```sh
pip3 install mmh3
```
